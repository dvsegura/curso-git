Introducción

Git,sistema de control de versiones moderno más utilizado del mundo.Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005

Presenta una arquitectura distribuida, es un ejemplo de DVCS (sistema de control de versiones distribuido, por sus siglas en inglés).

Configuración Inicial

        
          $ git --version 
          $ git config --global user.name "myName" 
          $ git config --global user.email "email"
          $ git config --global core.editor "code --wait" 
          $ git config --global -e  // consultar archivo de configuración

Configurar el core.autocrlf pues varia segun el Sistema Operativo desde donde estemos trabajando.

  • Window -> core.autocrlf true
  • Linux/Mac -> core.autocrlf input
            
                $ git config --global core.autocrlf true
